# 購物車網站
![image](https://github.com/weihung-1010/shopcar_project/blob/main/images/shop.png)
## 功能＆簡介
* 使用者可以透過按鈕切換表單分頁
* 透過按鈕上一步＆下一步掌控目前進度
* 使用者可以透過購物籃中的「＋」和「−」按鈕調整購買數量
* 消費總金額會根據使用者的購買數量而改變
* 所選的運送方式可以改變運費金額
* 在376螢幕尺寸以下，導覽列會縮成漢堡排

## 使用
在指定資料夾位置，將專案 clone 到本地
```
https://github.com/weihung-1010/shopcar_project.git
```
